# 🏠 Property Sales Analysis

A data analysis project exploring synthetic property sales data using Python, pandas, and visualization libraries.

## 🔧 Tools & Libraries
- Python 3.x
- pandas, numpy
- matplotlib, seaborn
- faker (for generating mock data)

## 📁 Project Structure
- data/: Contains the CSV file
- src/: Python script for generating fake data
- notebooks/: Jupyter Notebooks for EDA
- visuals/: Saved charts/images (optional)
